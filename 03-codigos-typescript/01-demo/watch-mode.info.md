# Whatch Mode

## Monitorización de un archivo:

```js
tsc nombreArchivo.ts --watch
tsc nombreArchivo.ts -w 
```

## Detener monitorización de un archivo:

```Terminal
Control + c
```

## Inicializar proyecto TypeScript

```Terminal
tsc -init
```

## Monitorización de TODOS los archivos del proyecto:

```Terminal
tsc *.ts -w
```
